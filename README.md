## Project Overview

Connectly is a Django-based project created to practice backend development concepts.
The project focuses on creating and managing users and posts while applying Django models, views, URL routing, migrations, and testing.

## Setup Instructions

1. Clone the repository and navigate into the project folder.
2. Activate the virtual environment.
3. Run migrations and start the development server.

* python manage.py makemigrations
* python manage.py migrate
* python manage.py runserver
